# Footprint-level Building height Estimation
Using Random Forest (RF) and SHAP to estimate footprint-level building height. 
## Packages
- memopy
- pandas
- geopandas
- numpy
- rasterio
- sklearn
## Data
- FABDEM
- AW3D30 DSM
- OpenStreet Map
- EUBUCCO
## Citation
